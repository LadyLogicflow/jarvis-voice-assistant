## Getting Started

### Prerequisites

- Python 3.11+
- pip or uv package manager

### Installation

```bash
# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # Linux/macOS

# Install dependencies
pip install -r requirements.txt

# Run the application
python -m src.main
```

### Running Tests

```bash
pytest tests/
```
