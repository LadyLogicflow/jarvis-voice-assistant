JARVIS — Funktionsübersicht

Catrin Essberger-Brenscheidt · Stand: 6. Mai 2026

Sprache + Interaktion

  • Web-UI am Mac: Klatschen oder Tastenkombination starten Jarvis. Spracherkennung via Web Speech API (Deutsch).
  • 30-Sekunden-Follow-up-Fenster: Nach jeder Jarvis-Antwort kannst du 30 s direkt antworten OHNE „Jarvis"-Wake-Word voranzustellen.
  • Telegram-Bot: Voice-Notes oder Text. Lokales Whisper transkribiert, Antworten als Voice-Note + Text-Caption zurück.
  • Anrede + Begrüßung: Pro Anfrage zufällige Anrede aus Pool (Catrin/Caterina/Madam) + tageszeit-passende Floskel (Guten Morgen / Guten Tag / Guten Mittag / Guten Nachmittag / Guten Abend).

Mail-Monitor (Apple iCloud)

  • Polling-basiert (60 s), klassifiziert eingehend in werbung/info/handlungsbedarf via Claude Haiku.
  • Auto-Triage: Sender-Regeln pflegbar in mail_triage_rules.json — deine 5 Regeln (coolblue, dihag, amazon, scansnap, hellomed) sind drin.
  • Heuristiken: Bounce → Junk, Pakete → DHL, Reise → Reise, Newsletter → Werbung.
  • Hellomed-Sonderfall: bei Anhang Auto-Forward an getmyinvoices.net via SMTP.

Mail-Decision-Tree

  • Direktbefehle nach Mail-Notification: vorlesen / fass zusammen / antworten / aufgabe / ignorieren.
  • Antwort-Workflow: Jarvis schlägt proaktiv eine Antwort vor (nutzt business_context.md mit deinen Standard-Sachverhalten), liest vor, fragt nach Freigabe. Iterativ „Änderung wie folgt" bis „freigeben" → in Apple-Drafts. Sendet NIE selbst.
  • Bei fehlendem Kontext: NEED_INPUT-Marker → Jarvis fragt nach Eckpunkten statt zu erfinden.
  • Kalender-Einladungen: ICS-Anhang erkannt → „Termin am X eintragen?" → Google Kalender.

Personen-Datenbank + Kontakte

  • Apple Kontakte.app als Read+Write-Source via AppleScript.
  • Auto-Extract bei unbekanntem Sender: Claude rät Anrede, Funktion, Organisation aus dem Mail-Inhalt → Vorschlag „Max Mueller (Geschäftsführer X) anlegen?".
  • Email-Drift-Detection: Mail vom bekannten Namen aber neue Adresse → Update-Vorschlag.
  • Telefon-Drift-Detection: Mail mit neuer Nummer in Signatur → Ergänzungs-Vorschlag.
  • Telefon-Wahl: „Rufe Mueller an" → Auswahl wenn mehrere Nummern, dann FaceTime/iPhone-Continuity.

Aufgaben + Termine

  • Todoist: gescoped auf deine 3 Projekte (HILO-Section, DIHAG, Privat). Aufgaben anlegen, abschließen, anzeigen. Bei „abschließen" nur deine eigenen, mehrfache Treffer fragen nach.
  • Google Kalender: Termine anzeigen + anlegen.
  • Mail → Aufgabe: aus Mail in Todoist-Inbox mit kontextbasierter Beschreibung (kennt Funktion der Person).

Gedächtnis

  • Merk-dir-Befehl:
    • „Merk dir: ich trinke morgens schwarzen Kaffee" → Vorliebe
    • „Merk dir zu Mueller: Bilanz braucht's bis Freitag" → Notiz an Mueller verknüpft + Aufgaben-Vorschlag
    • „Merk dir Anrede für Mueller: Du, Max" → setzt direkt die Anrede
  • Recall: „Was hatte ich zu Mueller?" → Volltext-Suche über Personen-Notizen, allgemeine Notizen, Todoist-Tasks, Conversation-Verlauf.

Briefings

  • Morgenbriefing bei Activate vor 11 Uhr: Datum + Wetter + Termine + Aufgaben + Steuer-News + Politik (in zufälliger Reihenfolge).
  • Proaktive Updates 12:30, 16:00, 18:00.
  • Politik-Brief: 3 wichtigste News (Inland + Wirtschaft Mix) + optional Goodnews am Ende.
  • Wochenausblick: Sonntag 18:00 automatisch + on-demand. Alle Termine der nächsten 7 Tage + alle offenen Aufgaben einzeln aufgelistet.
  • Steuer-News: BMF/BFH-RSS-Feeds zusammengefasst.

Quiet Hours + Sicherheit

  • Quiet Hours: Telegram still ab 21:00, Mac-UI ab 22:00, beide bis 7:00.
  • WebSocket auf Localhost gebunden — niemand im WLAN kann Jarvis ansteuern.
  • FileVault aktiv, Dateirechte 600 auf .env / config.json, sicherer .env-Parser im Launcher.

Werkzeuge

  • Browser-Suche (DuckDuckGo) + URL öffnen + Webseite zusammenfassen.
  • Bildschirm-Screenshot + Beschreibung via Claude Vision.
  • macOS Notes.app-Notiz anlegen.
